### Word Guesser
*   **Files:**
    * `word-guesser/word-guesser.c` 
    * `word-guesser/dictionary.txt`
    * `word-guesser/words.txt`
*   **Description:**
    - `word-guesser` directory contains `word-guesser.c`, the code for the Word Guesser game.
    - `dictionary.txt` contains all the valid 5-letter words allowed in the guesses.
    - `words.txt` contains over 3000 common words from which one word will be selected randomly as the key.
*   **Usage:** Guess the five-letter word using clues provided after each guess. Each guess must be a valid 5-letter word in the dictionary. After each guess, clues will be provided for each letter of the word, i.e., 2 denotes the letter is at the correct spot, 1 denotes the letter is at the wrong spot, and 0 denotes the letter is not in the word.
